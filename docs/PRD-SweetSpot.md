# PRD: SweetSpot
Sweet spot is a social web app designated towards sharing anything related to desserts.
It allows users to share links to articles, videos, or any other free online resource that lets other discover new sweet recipes and reviews. It also allows other users to comment and vote on those resources. 

# Experience 

## Posts list
The landing page of the site is a list of links that are horizontal to each other : 

    - Each link has a title given by the poster
    - Each link has a score based on its popularity and age
    - Linkes are ordered based on their scores
    - Each link has a list of comments left by users

## Popularity 

The popularity of a post is calculated based on: 

    - Number of votes
    - Number of views
    - Number of comments

## Members and Guests
Users are categorized into two: 

**Guests:** Users who aren't signed in, they are able to : 
- Browse The list of posts
- Filter based on tags, popularity, age, language

**Members:** Signed in users, they are able to do everyting that guests can do as well as: 
- Add or delete comments
- Upvote posts or comments
- Add or delete posts
 
## Tags
A post can have a set of tags that describe the dessert or the post

Some tags include:

**Dessert:** dessert type, main flavor, occasion, homeland
    
**Post:** Type, language, creator


## Comments
for any given post, comments can either be sorted chronologically, reverse-chronologically or based on upvotes 

# Market
Hybrid experience between forms sites such as Hackernews or Reddit and recipe sharing sites such as Allrecipes and Cookpad
